
本程序是做一个基于MFC的飞机大战单机版游戏。  
游戏主要角色有战机、敌机、战机导弹、敌机子弹、客机、保护膜星星及保护膜，  
其中战机可升级，有五种形态，导弹可随战机等级升高数目增多，形态变化。
  
  
  关键技术：  
文字显示、  
透明贴图技术、  
图片链表实现爆炸和保护膜动画、  
键盘消息响应、  
定时器的使用、  
双缓冲技术、  
碰撞检测技术、  
内存释放。  
  
  
  功能需求：
战机数量为1，键盘控制，可以随杀敌数增加而升级，  
有五个等级，五种形态，升级后子弹数目增加，类型变化。  
敌机出现时间、位置、子弹随机，随关数增加，生命增加。  
子弹、导弹、敌机碰撞后会消失，战机生命数为零时游戏结束。  
随机产生客机，击中客机不同位置会减少不同分数与生命值。  
所有敌方物体和导弹进行越出游戏区域则进行销毁，释放内存。  
为游戏提供相关背景，不同关卡之间可实现不同界面背景的转换。  
战机碰随机产生的星星会产生一定时间的保护膜。  
游戏结束后玩家可以在设置栏选择重新开始。  
实现了双缓冲技术进行解决刷屏屏闪的问题。 

  环境需求：VS2010 + windows 7
